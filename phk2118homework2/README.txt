-- written.txt --
Contains solutions to written problems.




-- FindPalindromes.java --
Finds and prints all palindromes in palindromes.txt (assuming palindromes.txt is placed in the same directory).


Compile using any java compiler, and run using a JVM.




-- MyStack.java --
Defines the MyStack class used in FindPalindromes.java and TwoStackQueue.java. This class uses the LinkedList
from java.util to create a stack, which pushes and pops from the front of the list. It also contains other useful methods.


Compile using any java compiler.




-- TwoStackQueue.java --
Defines the TwoStackQueue class used in Program2.java. This class uses 2 stacks (defined in MyStack.java) to build
a generic queue.


Compile using any java compiler.




-- Program2.java --
Test program to test TwoStackQueue.java. Enqueues integer elements, and then prints to screen the size of the
queue at various times. At the end, it dequeues all elements and prints them to the screen.


Compile using any java compiler, and run using a JVM.




-- Program2.txt --
Discusses big-O running times for enqueue and dequeue operations in TwoStackQueue.java.
